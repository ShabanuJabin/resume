Projects

Malabis

“Malabis” is an online tailoring mobile application that allows users to book for stitching. This system also allows tailoring shop to automate processes in managing different transactions in their shop. The system is design to automate transactions such as managing customer’s information, taking customer orders, checking on the progress of the garments, payments and all other transactions that takes place in a tailoring shop. Malabis aimed to automate the tailoring sector which is manually maintained. After the automation, better services and good keeping of records, transaction, design, and also paperless environment. This project is developed using flutter as front end framework and firebase as back end.


Smart Eye

In this project, facial recognition in CCTV image is done to be a base for decision making to produce automated, effective and efficient integrated system. This CCTV image processing provides two outputs, face detection information and face identification information. We implement this idea for the college. In the system college staff can monitor quarrel between students, and identify the students included in the quarrel by using face recognition system. Information is the backbone of any organization. Therefore it has to be made available to all times to ensure proper decision making and towards this end, information has to be accurate, currently, timely, relevant and usable. This project is developed using HTML and CSS as front end and Python as back end and its database is MySQL .
